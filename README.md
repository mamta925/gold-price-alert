# Gold Price Alert

Automated **trailing historical low** alerts for Gold Futures (`GC=F`) — headless Python job on **[SnapDeploy](https://snapdeploy.dev)** with Email + WhatsApp notifications.

## Deployment (SnapDeploy + GitHub Actions)

| Component | Role |
|---|---|
| **SnapDeploy** | Hosts the Python container (free tier: 512 MB, auto-sleep/wake, GitHub deploy) |
| **GitHub Actions** | Fires daily at **08:30 IST** → `POST` to SnapDeploy `/run` (wakes container) |
| **This repo** | Source code; connect to SnapDeploy via GitHub integration |

### Setup checklist

1. Push code to GitHub.
2. [SnapDeploy](https://snapdeploy.dev) → **New Container** → connect this repo.
3. Set env vars in SnapDeploy dashboard (SMTP, Twilio, `CRON_SECRET`, etc.) — see `prd.md` Section 8.
4. Add GitHub Actions secrets: `SNAPDEPLOY_APP_URL`, `CRON_SECRET`.
5. Enable `.github/workflows/daily-alert.yml` (created during `/speckit-implement`).

**Free tier usage:** 1 wake/run per day ≪ 10 daily limit. No credit card required.

## Spec-Driven Development (Spec Kit)

This project uses [GitHub Spec Kit](https://github.com/github/spec-kit) with **Cursor Agent** integration.

### Artifact map

| Artifact | Path | Purpose |
|---|---|---|
| Product requirements (human-authored) | `prd.md` | Source requirements, locked decisions |
| Constitution | `.specify/memory/constitution.md` | Project dos/don'ts — governs all AI work |
| Feature spec | `specs/<feature>/spec.md` | Generated by `/speckit-specify` |
| Implementation plan | `specs/<feature>/plan.md` | Generated by `/speckit-plan` |
| Tasks | `specs/<feature>/tasks.md` | Generated by `/speckit-tasks` |
| Cursor skills | `.cursor/skills/speckit-*` | Slash-command workflows |

### End-to-end workflow

Run these **in order** in Cursor Agent chat:

```
1. /speckit-constitution   → Lock dos/don'ts (do this first)
2. /speckit-specify        → Create spec from prd.md (or describe feature)
3. /speckit-clarify        → Optional: resolve ambiguities before planning
4. /speckit-plan           → Technical plan (Python, yfinance, cron, etc.)
5. /speckit-checklist      → Optional: quality checklist
6. /speckit-tasks          → Actionable task breakdown
7. /speckit-analyze        → Optional: cross-artifact consistency check
8. /speckit-implement      → Execute tasks and build code
9. /speckit-converge       → Find remaining gaps after implementation
```

**Gate rule:** Do not run `/speckit-implement` until you approve `spec.md`, `plan.md`, and `tasks.md`.

### Quick start (constitution → spec)

After answering the dos/don'ts questionnaire, run:

```
/speckit-constitution Use prd.md and my answered dos/don'ts as input.

/speckit-specify Build the Gold trailing-low alert engine described in prd.md.
Focus on what/why only — no code yet.
```

### Prerequisites

- Python 3.11+
- [uv](https://docs.astral.sh/uv/) (for Spec Kit CLI)
- Cursor Agent (skills installed under `.cursor/skills/`)

Re-init Spec Kit (if needed):

```bash
uvx --from git+https://github.com/github/spec-kit.git specify init --here --integration cursor-agent --force
```

### Secrets

Copy `.env.example` → `.env` (never commit `.env`).
